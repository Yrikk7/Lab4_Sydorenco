Додатки Google

Google Переводчика
С помощью этого расширения, разработанного командой Google Переводчика, можно быстро переводить веб-страницы.
uBlock Origin:
uBlock₀ — эффективный блокировщик: он использует меньше оперативной памяти и меньше нагружает ЦП, при этом используя больше фильтров, чем другие популярные блокировщики.
Speedtest by Ookla:
Проведите тест скорости прямо с панели инструментов Google Chrome, чтобы быстро и без перерыва проверить производительность Интернета. Измерьте, насколько быстро загружаются посещаемые вами страницы с помощью Web Speed.
Tab Manager Plus for Chrome:
Быстро находите открытые вкладки, просматривайте все окна в одном представлении, находите дубликаты и ограничивайте количество вкладок для каждого окна. Лучший менеджер вкладок для Chrome.
Autofill:
Самое популярное расширение автозаполнения, повышающее производительность с 2010 года.
RoPro:
Идеальный инструмент для игроков и трейдеров. RoPro добавляет в Roblox десятки полезных функций.
MyBib: Free Citation Generator:
Автоматически создавайте стиль APA, формат MLA и цитаты в стиле Гарвардских ссылок с помощью нашего генератора цитат для Chrome.
